## Pulsar Admin Manager

front end base on [vue-element-admin](https://panjiachen.github.io/vue-element-admin/#/dashboard).

back end base on spring boot.

### Feature

* Tenants manage
* Namespace manage
* Topics manage
* Subscription manage
* Function manage
* Source manage
* Sink manage
